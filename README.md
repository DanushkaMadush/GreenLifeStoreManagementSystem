## Overview

**GreenLifeStoreManagementSystem** is a feature-rich **Windows desktop application** crafted to optimize store operations, inventory management, and customer interactions within the **.NET Framework**.

It provides a solid foundation for building scalable and maintainable store management solutions.

## Why GreenLifeStoreManagementSystem?

This project aims to simplify complex store workflows through a **modular, user-friendly interface**.

Core features include:

-  **Role-Based User Management** — Supports different user roles (customers, admins) with secure authentication
-  **Intuitive UI Components** — Seamless navigation with dedicated forms for login, home, and management tasks
-  **Localization & Configuration** — Built-in support for internationalization and customizable settings
-  **Secure Authentication** — Ensures safe user login and session management
-  **Modular Architecture** — Organized project structure for easy maintenance and scalability

## Getting Started

### Prerequisites

- **Programming Language**: C#
- **Package Manager**: NuGet
- **.NET Framework** compatible development environment (Visual Studio recommended)

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/DanushkaMadush/GreenLifeStoreManagementSystem.git
   ```

2. Navigate to the project directory

   ```bash
   cd GreenLifeStoreManagementSystem
   ```

3. Restore NuGet dependencies

   ```bash
   dotnet restore
   ```

### Usage

Build and run the application:

```bash
dotnet run
```

### Testing

The project uses a test framework (most likely **xUnit**, **NUnit**, or **MSTest** — check project files for exact framework).

Run the test suite with:

```bash
dotnet test
```
